Chapter 5. Installation

Table of Contents

Currently the Tools support Eclipse 3.6 (Helios) and above with the corresponding version of WTP. Eclipse 3.7 (Indigo) is recommended. The Eclipse IDE for Java EE Developers is recommended for most users.

The Virgo Tools, from one of the following update sites depending on your version of Eclipse. Releases are, in general, more stable than milestones whereas nightly builds are not for the faint of heart.

  • http://download.eclipse.org/virgo/milestone/tooling

  • http://download.eclipse.org/virgo/snapshot/tooling
